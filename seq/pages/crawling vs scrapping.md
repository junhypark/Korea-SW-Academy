- # JS or JQuery
  collapsed:: true
	- 대부분의 메인 웹사이트들은 데이터를 모두 HTML에 올리지 않는다
	- 이때, 개발자 도구에서 network로 이동해 어떤 JS 혹은 JQuery가 움직이는지 확인해야한다
	- RESTful 함수들은 GET의 경우 주소와 파라미터 값을 통해 데이터가 움직이기 때문이다
